# MessagePolish

Adds a chat bar button that improves your current message with AI and copies the improved result to your system clipboard.

Supports:
- OpenAI, Anthropic, and Google
- Per-channel style switching
- Tasteful markdown when it helps

## Install

If you already have a Vencord user plugin development setup, put this plugin in your Vencord `src/userplugins/` directory.

If you have never set up a Vencord user plugin before, set up a Vencord development environment first, then place this plugin in your `src/userplugins/` folder.
